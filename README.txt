Risk of Rain 2 — static HTML/CSS project

How to open:
1. Extract the ZIP archive.
2. Open index.html in a browser.

Pages:
- index.html
- survivors.html
- artifacts.html
- items.html
- gallery.html

Project features:
- external CSS file
- navigation between all pages
- local SVG graphics and external URL images
- responsive layout with media queries
- Flexbox/Grid layout
- hover effects and transitions
- inline SVG graphic
- HTML image map
- footer with author information on every page
